Experiência imersiva de visualização de vídeo

O que é o Efeito Ambilight?

O efeito Ambilight (Ambient Lighting) foi originalmente desenvolvido pela Philips para suas TVs. Ele analisa o conteúdo sendo exibido na tela e projeta cores correspondentes além das bordas do display, criando uma experiência mais imersiva e reduzindo a fadiga ocular durante visualizações prolongadas.

Nossa implementação do Ambilight para web usa tecnologias modernas para trazer essa experiência para qualquer vídeo em seu navegador. O sistema analisa as cores nas bordas do vídeo em tempo real, dividindo-as em zonas configuráveis, e estende essas cores além do player de vídeo.

Como Funciona

O Ambilight Player utiliza uma arquitetura cliente-servidor para proporcionar o efeito Ambilight em tempo real:

  1. Processamento de Vídeo: O backend Python utiliza OpenCV para analisar cada frame do vídeo e extrair as cores das bordas.
  2. Divisão em Zonas: As bordas são divididas em zonas configuráveis (5-20 por lado) e a cor média de cada zona é calculada.
  3. Transmissão em Tempo Real: As informações de cor são enviadas para o frontend via WebSockets.
  4. Renderização do Efeito: O JavaScript aplica as cores como efeitos CSS (box-shadow) em torno do player de vídeo.
  5. Otimizações: O sistema inclui cache de frames e limitação de taxa para garantir desempenho suave mesmo em dispositivos menos potentes.

Uso

Para utilizar o Ambilight Player, basta fazer upload de um arquivo de vídeo suportado ou selecionar um vídeo do histórico. As configurações do efeito Ambilight podem ser ajustadas em tempo real através do painel de ajustes rápidos ou na página de configurações.

O player é otimizado para uso em navegadores modernos como Chrome, Firefox, Safari e Edge.
